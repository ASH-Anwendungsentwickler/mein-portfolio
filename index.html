Meine Projekte

Migrationtool (Tool)

Das Migrationtool ermöglicht das Laden, Bereinigen und Verarbeiten von Datensätzen aus verschiedenen Quellen. Mithilfe regulärer Ausdrücke können Daten flexibel angepasst werden. Integrierte, vorkonfigurierte SQL-Abfragen erleichtern auch Anwendern ohne SQL-Kenntnisse den Zugriff auf die Daten. Abschließend lassen sich die Datensätze direkt in die Datenbank einlesen oder aus ihr exportieren.

Technologie:

  • C#
  • |
  • Windows Forms

TicTacToe AI (Spiel)

Das TicTacToe AI Spiel hebt das klassische 3x3-Feld auf ein neues Niveau. Eine intelligente KI bewertet alle möglichen Züge, um eine stets fordernde Gegnerin zu sein. Unterschiedliche Schwierigkeitsstufen garantieren unterhaltsame Partien für Einsteiger und Profis gleichermaßen. Das aufgeräumte WPF-Interface sorgt für eine intuitive Bedienung, während C# im Hintergrund die Spielmechanik steuert.

Technologie:

  • C#
  • |
  • WPF
